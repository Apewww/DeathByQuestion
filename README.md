# DeathByQuestion

DeathByQuestion adalah aplikasi kuis sederhana berbasis Java yang menyajikan pertanyaan kepada pengguna dan mengevaluasi jawaban yang diberikan. Proyek ini dibuat sebagai latihan penerapan konsep Object-Oriented Programming (OOP) serta penggunaan struktur data di Java.

Aplikasi ini cocok digunakan sebagai media pembelajaran, latihan soal, maupun contoh implementasi program kuis berbasis konsol.

---

## Fitur

- Menampilkan pertanyaan kuis
- Menyediakan pilihan jawaban
- Menentukan jawaban yang benar
- Struktur kode berbasis class (OOP)
- Mudah dikembangkan untuk fitur lanjutan

## Cara Menjalankan Program

1. Clone repository  
   git clone https://github.com/Apewww/DeathByQuestion.git

2. Masuk ke folder project  
   cd DeathByQuestion

3. Compile program  
   javac src/deathbyquestion/*.java

4. Jalankan program  
   java deathbyquestion.Main

Pastikan Java JDK sudah terpasang di komputer.

---

## Gambaran Implementasi

Program menggunakan class Question untuk menyimpan teks pertanyaan, daftar opsi jawaban, dan indeks jawaban yang benar. Pendekatan ini membuat pengelolaan soal lebih terstruktur serta memudahkan pengembangan fitur tambahan di masa depan.
